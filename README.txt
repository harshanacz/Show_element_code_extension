# Extension Installation Guide

1. Extract the contents of this zip file to a directory.
2. Open Microsoft Edge.
3. Run the `install_extension.bat` script by double-clicking it.
4. This will open the Edge Extensions page.
5. Enable "Developer mode" by toggling the switch on the top-right corner.
6. Click "Load unpacked" and select the directory where you extracted the files.
7. The extension should now be loaded and ready to use.
